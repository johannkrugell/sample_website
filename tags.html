Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell
div div a section in the document style with css
header header a header of a document
ol ol a order list
ul ul unordered list
li li list item

Inline Elements

Tag Name Purpose Example Results
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong make text strong <strong>at least a billion people</strong> at least a billion people
a anchor make hyperlink <a href="https://learnenough.com/" >Learn Enough </a> Learn Enough
img image include an image <img>src="https://bit.ly/1MZAFuQ" alt="Michael Hartl"</img>
code code make text monospaced font (type writer) <code>type writer </code> type writer
span span used the markup part of text <span>styled text </span> styled text

Document Tags

Tag Name Purpose
<html> html The <html> element is the root element and it defines the whole HTML document.
<head> head The <head> element is a container for metadata
<body> body The <body> element contains all the contents of an HTML document
<title> title The <title> tag defines the title of the document
<meta> meta The <meta> tag defines metadata about an HTML document
<div> div A <div> section in a document that is styled with CSS
<blockquote> blockquote A <blockquote> is used to quote a section from another source
